# Kario Media PM
Aplicativo Web que pueda mostrar el panel de gestionamiento de proyectos y sus módulos necesarios para la empresa “Kario Media”,
la cual realiza licitaciones públicas y privadas de varios nichos de negocio.

## Descripción
La página  Web cuenta  con  6 opciones  principales:

1.  **Añadir** : Incluye un formulario que permite añadir nuevos indicadores.
2.  **Refrescar** : le permite al usuario recargar nuevamente la página .
3. **Eliminar** : Incluye un formulario que le permitira eliminar al usuario algunos datos.
4. **Reportar** : incluye un formulario que  permite  al usuario reportar alguna falla dntro de su indicador.
5. **Ayuda**:  incluye opciones  que buscan  aportar  más información al usuario.
6. **Perfil** : le permite al usuario  revisar  el  perfil  y asi mismo cuenta con la opción de cerrar sesión.

## Característica de Diseño

* Diseño responsive  utilizando medidas vw (viewport width) para las dimensiones.
* Barra de navegación pagina principal que cuenta con las opciones (Añadir,      Refrescar, Eliminar , Reportar , Ayuda y Perfil (Log-In y Log-Out)).
* Uso de icons e  imagenes para fondos  y  estética de la página.
* Uso de animaciones para algunos menús.
* Menú desplegables para cada una de las opciones de navegación.
* Media queries en cada una de las páginas.
* Transiciones automaticas de algunas páginas .
